conky配置

2016-06-04
Linux      software

0x00 效果

使用软件conky可以在linux上看到系统的运行状态，效果如下：

0x01 安装与配置

  1. 先安装conky：
    sudo apt-get install conky

  2. 在/home/username/下创建文件.conkyrc

  3. 使conky开机自启
    在/etc/profile最下面添加：/usr/bin/conky &

0x03 配置文件.conkyrc内容如下
